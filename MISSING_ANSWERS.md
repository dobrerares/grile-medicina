## How to patch

Add a JSON file in `patches/` with this structure:

```json
{
  "target": "Biologie 2011 UMFCD.json",
  "patches": [
    {"test_title": "TESTUL NR. 13", "question_number": 1, "correct_answer": "B"},
    {"test_title": "TESTUL NR. 13", "question_number": 2, "correct_answer": "A"}
  ]
}
```

For complement_grupat (CG), use the GRUPAT answer letter (A–E) which maps to statement combinations:
- A = 1,2,3
- B = 1,3
- C = 2,4
- D = 4
- E = 1,2,3,4

To override a wrong existing answer, add `"force": true` to the patch entry.

Then re-run: `nix develop --command bash -c "python scripts/parse.py && python scripts/patch_answers.py && python scripts/deduplicate.py && python scripts/merge.py"`
